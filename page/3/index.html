从手写 UI 到 AI 拼 UI：游戏界面开发方式的三次变化

  这几年做游戏开发，一个很明显的变化，就是 UI 的生产方式一直在变。十年前我在使用 cocos2d-x，那时候做界面基本就是“手搓”。按钮放哪、背景图怎么对齐、列表怎么排版、不同分辨率怎么适配，都要开发自己写代码处理。UI 不是不能做，而是成本很高，很多时间其实花在了重复劳动上。一个登录界面、一个背包界面，真正复杂的业务逻辑还没开始，光把节点摆好、响应接好，就已经消耗了不少精力。

  到了 2020 年以后，开始更多使用 Cocos Creator，情况就不一样了。Creator 把 UI 搭建从“写代码”变成了“可视化拼接”。节点层级、组件挂载、资源拖拽、布局调整，很多事情都可以直接在编辑器里完成。对开发来说，效率提升是非常明显的。以前要反复改坐标、改锚点、重新编译才能看到效果，现在在编辑器里拖一拖、点一点，立刻就能看到结果。某种意义上，这一代工具把 UI 开发门槛拉低了，也把很多机械性的工作从程序员手里拿走了。

  而现在，AI 又把这件事往前推了一步。它开始尝试“自动拼 UI”。也就是说，开发者不一定非要手动摆节点，也不一定非要自己一个控件一个控件去拖。你只要把需求描述清楚，AI 就有机会先给出一个初版界面，甚至把部分组件结构、样式层级、交互流程一并生成出来。从趋势上看，UI 的生产方式已经从“程序员自己实现”，走到了“工具辅助搭建”，再走到了“AI 参与生成”。

  不过我觉得，这里面最关键的一点不是 AI 会不会画界面，而是你有没有把足够准确的语料喂给它。这里的语料，不只是几句自然语言提示词，更重要的是和项目相关的 API、组件规范、命名规则、交互约束、页面结构，甚至是现有工程里的资源组织方式。AI 不是凭空理解你的项目，它生成出来的 UI 好不好，很大程度取决于你给它的信息是否完整。如果没有这些上下文，AI 能做的往往只是“看起来像那么回事”的 demo，而不是能真正接进项目里的 UI。

  所以我更倾向于把现在这个阶段理解成：AI 不是代替人来做 UI，而是把开发者的工作重心往前移了。以前我们主要在“实现”上花时间，现在更多是在“描述”和“约束”上花时间。你要先把接口定义清楚，把页面状态想明白，把组件边界整理好，再把这些内容交给 AI。谁的工程规范更清晰，谁的 API 设计更稳定，谁就更容易把 AI 用起来。

  从 cocos2d-x 到 Cocos Creator，再到今天 AI 开始拼 UI，我看到的不是某一种工具替代另一种工具，而是 UI 开发正在不断抽象、不断提效。手写代码解决的是“能不能做”，可视化编辑解决的是“能不能更快做”，AI 现在尝试解决的，则是“能不能先帮你做出第一版”。至于最后效果怎么样，还是取决于开发者对业务、工具和工程上下文的理解。

《中华古今注》：在名物之间保存古人的日常

  《中华古今注》这样的小书，最宜在夜深人静时翻读。它不厚，也不张扬，纸色已有旧意，版框方整，字口清楚，拿在手里先有一种温雅气。这样的书，不靠篇幅压人，也不以议论取胜，胜在一页页之间都藏着古人的生活痕迹。宫室城阙、车服器用、草木鸟兽，本是寻常名物，到了这类书里，却忽然都有了来历。

  今本《中华古今注》通常题作五代马缟撰。马缟在序中明言，晋人崔豹旧有《古今注》，见闻已广，只是尚有缺略，因此自己又“添其注，以释其义”。所以这书读来，很像一条文脉的续接，不是平地另起炉灶，而是在旧书基础上续作补苴，把前人留下的材料重新整饬，使其更便于后人寻绎。

  若从体例看，它近于古人的名物书。它并不专叙帝王功业，也不铺陈大段史事，而是从一个个名词入手，考其称谓，辨其沿革，间及制度、风俗与掌故。今天读到“宫”“阙”“城”“沟”这些字，往往只是纸上的概念；放回这类古书里，它们却都带着制度背景和生活气息。古代世界的轮廓，正是由这些细小处慢慢显现出来的。

  此书最可喜处，也恰在一个“杂”字。它所收甚博，却并不芜乱；篇幅虽短，却常有余味。往往只寥寥数语，便把一个名称的来源、一件器物的用途、一个旧说的转变说得大略分明。读者若把它当作闲书翻阅，自有兴趣；若把它当作入门，也能从中窥见传统名物学的门径。许多诗文中熟而不解的词，回头来查这类书，常会忽然明白几分。

  当然，《中华古今注》并非毫无疑义。后世目录家和考据家对《古今注》系统的真伪、增补、依托问题一直有讨论，认为其间并非一人一时之旧，而有层累附益之迹。正因如此，读它不必句句奉为定论。把它看成知识流转的痕迹，反而更有意味。抄录、转述、增补、重编，许多古书正是在这样的过程中慢慢定形；这本小书，也把那种传统学术生成的样子保存了下来。

  我向来偏爱这种小部头。它没有大书那样森严，却更近于案头清供。随手展开几叶，眼前浮出的也不只是几个术语，而是门阙、车辐、沟渠、衣冠、草木、禽兽所共同织成的一片古人日用世界。若从藏书的眼光看，这类书尤其耐人。它们未必显赫，却最能见时代纹理；未必堂皇，却常在细微处保存真气。

  这册《中华古今注》放在架上，也许并不最惹眼；但真抽出来细读，往往能生出绵长兴味。它提醒人，历史并不只在纪传、编年和大事记中，也藏在一个名称的解释、一件器物的辨析、一种习俗的追述里。能把这些细部保留下来的人，其实也替后来读书人留住了一部分古人的日常。藏书之乐，有时正是在这里。
